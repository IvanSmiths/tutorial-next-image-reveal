## Put Videos Inside Text with React Next.js

Repository of the [YouTube video]([https://youtu.be/QxVCigL1eyE](https://youtu.be/5Fxjt5ZNC00)) on how to play a video inside a text with React, Next.js and animating it with Gsap and it`s plugin ScrollTrigger.

## Stack used

- Next.js
- Gsap ScrollTrigger
- CSS
